# Lumen Studio

Ferramenta de autoria de conteúdo didático para professores. Crie, organize e exporte aulas e atividades em Markdown — gera slides `.pptx` via Marp e atividades em `.pdf`, tudo em uma janela.

---

## Instalação (Windows)

1. Acesse a página de [Releases](https://github.com/mateusflorez/senai-studio/releases/latest)
2. Baixe o arquivo `Lumen.Studio_x.x.x_x64-setup.exe`
3. Execute o instalador e siga as instruções

O app inclui todas as dependências de geração — não é necessário instalar Node.js, Chrome ou qualquer outra ferramenta.

Atualizações futuras são notificadas automaticamente dentro do próprio app.
