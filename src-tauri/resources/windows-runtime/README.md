# Runtime Windows

Esta pasta é preenchida pelo script `npm run prepare:windows-runtime`.

Conteúdo esperado:

- `node/` com `node.exe`
- `chromium/` com `chrome-headless-shell.exe`
- `node_modules/` com `@marp-team/marp-cli` e `markdown-it`
- `runtime-manifest.json` com as versões baixadas

O conteúdo pesado é gerado localmente e não deve ser versionado.
